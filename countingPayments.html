Counting Them Payments!

Counting payments is like cutting cake. Its easy for a few people but hard for a party.

The thing you need to count payments properly

  1. FedLoan login (or any servicer login) for payment history
  2. Most recent PSLF ECF. In some occactions you need all the past ones too.
  3. Members terms, yes all of them.
  4. Time!

In some occations I can get a PSLF case all done in about 10 minutes or less, BUT I have had cases that took me over 45 minutes because the payments were so jacked up.

You need to make sure of a few things.You need to make sure you are counting from employment start date unless is is older then the consolidation date. From there you will need to make sure the member is paying the full amount on time and no later then 15 days. Use the terms as a refernce.

When a member has been on a $0.00 payment there will be no payment history. Check notes to make sure member was never on ISD and or forbearance. If thats the case just count 12 payments per terms.
